# About


This repository contains **documented exercises for students** on climate data analysis and spatial risk assessment using Python.

It was developed in the context of the **ARKE research project** (*Adapting to the consequences of climate change for the Kolpingstadt Kerpen*) and uses the ARKE case study as a concrete example throughout the notebooks.
